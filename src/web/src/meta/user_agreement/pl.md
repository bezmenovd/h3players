## 1. Postanowienia ogólne

Niniejsza Umowa użytkownika („Umowa”) określa warunki korzystania ze strony internetowej h3players.com („Strona”) oraz udostępnionego za jej pośrednictwem interfejsu programistycznego (API).  

Administratorem Strony jest administrator h3players.com („Operator”).  
Adres kontaktowy: contact@h3players.com.  

Korzystając ze Strony lub API, użytkownik akceptuje warunki niniejszej Umowy.


## 2. Dostęp do Strony i API

- Użytkownik uzyskuje dostęp do Strony poprzez autoryzację za pomocą konta w odpowiednim lobby gry.  
- Operator przechowuje identyfikatory gracza (nick i ID) niezbędne do autoryzacji w ramach Strony oraz do wyświetlania statystyk gry.  
- API udostępnia dane związane z grą (nicki, identyfikatory w grze, informacje o rozegranych grach) wyłącznie w celu wyświetlania i analizowania statystyk.  
- Strona i API mogą być czasowo niedostępne z powodu prac technicznych lub awarii.


## 3. Prawa i obowiązki użytkowników

### Zabronione:
- publikowanie obraźliwych, nieprzyzwoitych lub niezgodnych z prawem materiałów, a także materiałów naruszających tożsamość innych użytkowników (np. oskarżenia);  
- publikowanie treści niezwiązanych z tematyką Strony;  
- korzystanie ze Strony lub API w celach oszustwa, ingerowania w działanie serwisu, ataków DDoS lub innych działań sprzecznych z prawem;  
- kopiowanie, modyfikowanie lub rozpowszechnianie danych Strony w celach komercyjnych bez zgody Operatora.

Korzystanie ze Strony i API we wszystkich innych przypadkach jest dozwolone pod warunkiem przestrzegania warunków niniejszej Umowy.


## 4. Własność intelektualna

- Wszystkie materiały na Stronie, w tym projekt, kod i treści, są własnością Operatora.  
- Dane związane z grą są udostępniane do wykorzystania w ramach Strony i API i nie mogą być używane poza tymi celami bez zgody Operatora.


## 5. Ograniczenie odpowiedzialności

- Strona i API są udostępniane „takie, jakie są”, bez gwarancji poprawności, kompletności lub ciągłej dostępności.  
- Operator nie ponosi odpowiedzialności za błędy, nieścisłości lub niedostępność danych.  
- Korzystanie z danych Strony odbywa się na własne ryzyko użytkownika.


## 6. Zakończenie korzystania

Operator może ograniczyć lub zablokować dostęp użytkownika do Strony lub API w przypadku naruszenia niniejszej Umowy.


## 7. Zmiana warunków

Operator zastrzega sobie prawo do wprowadzania zmian w niniejszej Umowie.  
Aktualna wersja publikowana jest na Stronie. Kontynuacja korzystania ze Strony po publikacji zmian oznacza akceptację zaktualizowanych warunków.


## 8. Prawo właściwe

Niniejsza Umowa podlega prawu Federacji Rosyjskiej.  
W przypadku użytkowników z innych jurysdykcji, w tym z Unii Europejskiej, mogą mieć zastosowanie również przepisy lokalne, takie jak przepisy dotyczące ochrony danych osobowych (np. GDPR).
