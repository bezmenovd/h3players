### 1. Общие положения

Настоящая Политика конфиденциальности определяет порядок обработки информации,
связанной с использованием сайта h3players.com (далее — Сайт).

Оператором Сайта является администратор сайта h3players.com.  
Контактный адрес для обращений: contact@h3players.com.

Оператор не осуществляет сбор данных, позволяющих напрямую идентифицировать
личность пользователя.


### 2. Состав обрабатываемых данных

В процессе работы Сайта обрабатываются следующие категории данных:

| Категория данных | Описание |
|-----------------|----------|
| Игровые данные | Никнеймы и идентификаторы игроков, используемые для отображения игровой статистики и авторизации в рамках Сайта |
| Технические данные | Хешированные IP-адреса (SHA-256) |
| Данные HTTP-запросов | Запрашиваемые URL, дата, время и длительность обработки запросов сервером |

Оператор не располагает сведениями, позволяющими сопоставить указанные данные
с конкретным физическим лицом вне рамок игрового контекста.


### 3. Цели обработки данных

Обработка данных осуществляется исключительно в следующих целях:

- отображение и анализ игровой статистики;
- обеспечение авторизации пользователей в рамках Сайта;
- обеспечение безопасности и стабильной работы Сайта;
- предотвращение злоупотреблений и автоматических атак;
- техническое администрирование и диагностика.

Данные не используются для установления личности пользователя,
а также для рекламных или маркетинговых целей.


### 4. Правовые основания обработки

Обработка данных осуществляется в соответствии с Федеральным законом
Российской Федерации от 27.07.2006 №152-ФЗ «О персональных данных».


### 5. Хранение и передача данных

Данные хранятся не дольше, чем это необходимо для достижения целей обработки.

Оператор может предоставлять доступ к игровым данным (никнеймы, игровые
идентификаторы, сведения о сыгранных игровых партиях) посредством программного
интерфейса (API) для целей отображения и анализа игровой статистики.

Указанные данные не содержат сведений, позволяющих напрямую идентифицировать
физическое лицо, и не сопровождаются техническими данными пользователей
(IP-адреса, данные HTTP-запросов и иные аналогичные сведения).

Передача иных категорий данных третьим лицам не осуществляется,
за исключением случаев, предусмотренных законодательством Российской Федерации.

Трансграничная передача данных не осуществляется.


### 6. Права пользователей

Пользователь вправе направить запрос:

- о прекращении обработки данных;
- об удалении данных, если их дальнейшее хранение не требуется
  для целей, указанных в настоящей Политике.

Запросы направляются по адресу: contact@h3players.com.


### 7. Заключительные положения

Оператор вправе вносить изменения в настоящую Политику.
Актуальная версия размещается на данной странице Сайта.
