### 1. General Provisions

This Privacy Policy defines the procedure for processing information
related to the use of the website h3players.com (the “Website”).

The controller of personal data is the administrator of the website h3players.com
(the “Operator”).  
Contact email: contact@h3players.com.

The Operator does not collect data that directly identifies a user,
such as name, government ID, residential address, phone number,
or email address.



### 2. Categories of Data Processed

The following categories of data are processed when using the Website:

| Data Category | Description |
|--------------|-------------|
| Game-related data | Player nicknames and in-game identifiers used to display game statistics and provide authorization within the Website |
| Technical data | Hashed IP addresses (SHA-256) |
| HTTP request data | Requested URLs, date, time, and server-side request processing duration |

The Operator does not possess information allowing the association
of the processed data with a real-world identity outside the game context.


### 3. Purposes of Data Processing

Data is processed solely for the following purposes:

- displaying and analyzing game statistics;
- providing user authorization within the Website;
- ensuring the security and stable operation of the Website;
- preventing abuse and automated attacks;
- technical maintenance and diagnostics.

The data is not used for user identification, advertising,
or marketing purposes.


### 4. Legal Basis for Processing

For users located in the European Union, data is processed on the basis of
the Operator’s legitimate interest in ensuring the security and proper
functioning of the Website, in accordance with Article 6(1)(f) of the GDPR.

For other users, data is processed in accordance with applicable data
protection laws.


### 5. Data Storage and Disclosure

Data is stored only for as long as necessary to achieve the purposes
of processing.

The Operator may provide access to game-related data (player nicknames,
in-game identifiers, and information about played games or matches)
via a programming interface (API) for the purpose of displaying
and analyzing game statistics.

Such data does not include information that directly identifies
a natural person and is not accompanied by users’ technical data
(such as IP addresses, HTTP request data, or similar information).

No other categories of data are disclosed to third parties,
except where required by applicable law.

No cross-border transfer of data is performed.


### 6. User Rights

Users may submit a request to:

- restrict or cease data processing;
- request deletion of data, where retention is no longer necessary
  for the stated purposes.

Requests may be submitted via email: contact@h3players.com.


### 7. Final Provisions

The Operator may update this Privacy Policy from time to time.
The current version is published on this page.
