## 1. General Provisions

This Privacy Policy defines the procedure for processing information related to the use of the website h3players.com (the "Website").

The controller of the Website is the administrator of h3players.com.  
Contact email: contact@h3players.com.

The Operator does not collect data that allows personally identifying the user.


## 2. Categories of Processed Data

During the operation of the Website, the following categories of data are processed:

| Data Category | Description |
|---------------|------------|
| Game Data | Nicknames and player identifiers; information about played matches |
| Technical Data | Hashed IP addresses (SHA-256) |
| HTTP Request Data | Requested URLs, date and time |


## 3. Purposes of Data Processing

Data processing is carried out exclusively for the following purposes:

- displaying game statistics;  
- enabling user authorization within the Website;  
- ensuring the security and stable operation of the Website;  
- preventing abuse and attacks;  
- technical administration and diagnostics.

Data are not used to identify the user personally or for advertising or marketing purposes.


## 4. Legal Basis for Processing

Data processing is performed in accordance with the Federal Law of the Russian Federation No. 152-FZ of 27.07.2006 “On Personal Data”.


## 5. Data Storage and Sharing

Data are stored no longer than necessary to achieve the processing purposes.

The Operator may provide access to game data via a programming interface (API) for the purpose of displaying game statistics.

The above data do not contain information allowing identification of a natural person and are not accompanied by users’ technical data.


## 6. Users’ Rights

Users have the right to submit a request for:

- cessation of data processing;  
- deletion of data if further storage is not required for the purposes stated in this Privacy Policy.

Requests should be sent to: contact@h3players.com.


## 7. Final Provisions

The Operator reserves the right to make changes to this Privacy Policy.  
The current version is posted on this page of the Website.
